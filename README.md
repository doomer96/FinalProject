Employee Information System

Davlet Beishenbekov

Employee Information System - Простая и удобная система для управления данными сотрудников. Проект создан для автоматизации учета персонала и хранения истории оценок эффективности

Основные функции:

Личные дела: Добавление, поиск и редактирование данных сотрудников

Учет должностей: Хранение информации об отделах и зарплатах

Оценка KPI: Фиксация результатов работы и отзывов

База данных: Надежное хранение всех данных в локальном файле
